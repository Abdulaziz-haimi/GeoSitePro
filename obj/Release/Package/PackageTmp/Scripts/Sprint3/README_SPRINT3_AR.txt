GeoSite Pro - Sprint 3
======================

بعد التأكد من نجاح Sprint 1 و Sprint 2، شغّل ملفات Sprint 3 بهذا الترتيب:

10_Create_Tables_Sprint3.sql
11_Create_Lookups_Permissions_Sprint3.sql
12_Create_StoredProcedures_Sprint3.sql
13_Verify_Sprint3_DB.sql

ما يضيفه Sprint 3:
- صفحة النتائج المعملية LabResults.aspx.
- ربط النتيجة بالمشروع والجسة والعينة.
- أنواع الاختبارات المعملية: الرطوبة، التحليل المنخلي، حدود أتربرج، CBR، UCS، وغيرها.
- حالات النتيجة: بانتظار الاختبار، قيد التنفيذ، مكتملة، تمت المراجعة، معتمدة، مرفوضة.
- صلاحيات عرض/إضافة/تعديل/حذف/اعتماد النتائج المعملية.
- تحديث عدادات Dashboard و Project Details لتشمل نتائج المختبر.

ملاحظة مهمة:
إذا كانت قاعدة البيانات عندك قديمة أو سبق تشغيل سكربتات مختلفة، شغّل أولًا ملفات Sprint 2 FIXED التي عالجت الأعمدة الناقصة، ثم شغّل Sprint 3.
